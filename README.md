# Børnebank - Simpel Prototype

## Hvordan det virker
1. Opret en Google Sheet og del den som "Alle med linket kan se"
2. Indsæt Sheet URL i appen
3. Start med at bruge den!

## Google Sheet Skabelon
Opret et regneark med denne struktur:

### Sheet 1: "Børn"
| A | B |
|---|---|
| Navn | Saldo |
| Anna | 50 |
| Mikkel | 25 |

### Sheet 2: "Transaktioner"
| A | B | C | D |
|---|---|---|---|
| Dato | Barn | Type | Beløb |
| 2026-03-18 | Anna | Lommepenge | 20 |
| 2026-03-18 | Mikkel | Bonus | 10 |

## Næste skridt
Filip skal oprette Google Sheet først!
